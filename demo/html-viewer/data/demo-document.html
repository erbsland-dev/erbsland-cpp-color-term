HTML Renderer Demo

This demo shows how the HTML renderer can display a structured document inside a scrollable terminal viewer. Use the cursor keys to move line by line, page up and page down to jump, press s to cycle between document styles, and press q when you want to quit.

What This Demonstrates

  • Headings, paragraphs, emphasis, strong text, and links.
  • Bullet lists, numbered lists, and nested list indentation.
  • Definition lists and code blocks with preserved line breaks.
  • A full-screen viewer with a header, footer, scrolling viewport, and switchable styles.

Nested List Example

  1. Load an HTML file.
  2. Render it to a terminal buffer.
    • Resize the layout when the terminal width changes.
    • Keep the rendered document in a scrollable view.
  3. Navigate with the keyboard.

Definition List

HtmlRenderer
The public API that turns HTML into terminal text or cursor-writer output.
TextNodeRenderer
The internal renderer that walks the parsed document tree.
CursorBuffer
A buffer that stores the rendered document and can grow vertically.

Code Block

auto buffer = std::make_shared<CursorBuffer>(Size{80, 1});
text::HtmlRenderer{html}.renderTo(buffer);
view.setContent(buffer);

Links render as link text only, for example erbsland.dev. The source file for this document lives in the demo directory, and you can pass your own HTML file as the first command-line argument.
