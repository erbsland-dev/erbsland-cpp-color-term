UI Sections

ui-sections is an interactive demonstration of ui::Sections.

Run it from the build directory with:

$ ./demo-apps/ui-sections

The demo lets you toggle a middle section, show or hide the trailing separator, and cycle the right-side title text. It is a compact reference for runtime visibility changes and section metadata updates.
